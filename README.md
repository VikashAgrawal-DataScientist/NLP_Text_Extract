# NLP_Text_Extract
Script to extract useful information from the given text

1) For "UsingOpenAI.py"

Need OPENAI API KEY to run this file, Please create an .env file, keep the OPEN_API_KEY="" and run the script.

2) For "Using_Spacy.py"

Please uncomment this line to download the model, 
# spacy.cli.download("en_core_web_sm")

This will save the model locally and script can executed successfully.
